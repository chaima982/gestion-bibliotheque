# 📚 Library Management System (MERN Stack)

## 📌 Description
This project is a full-stack web-based Library Management System developed using MongoDB, Node.js, Express, and React.

The application allows users to manage books through a modern and interactive interface connected to a RESTful API.

## 🚀 Features
- Add new books
- Edit existing books
- Delete books
- View and search book list
- REST API integration
- Real-time data updates

## 🛠 Technologies Used
- MongoDB (Database)
- Express.js (Backend Framework)
- Node.js (Server)
- React.js (Frontend)
- Axios (API requests)
- Mongoose (MongoDB ODM)

## 📂 Database
Database: MongoDB  
Collections: books

## ⚙️ Project Architecture
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- API: RESTful API
